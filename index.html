右键菜单新建文件添加更改

晚上打算写文章时，发现右键菜单里面没有新建md文件的快捷操作。就想着添加右键新建md文件。在网上查找了一下，前面几个方法，都只是说明了如何操作。并没有讲原理。知其然不知其所以然，往往出现了错误，但却不知道怎么该。查询一番后，大致明白了一些原理。记录下来，希望帮助到其他人。

原理:

右键菜单的修改本质上是修改注册表。

方法：

修改注册表的方法有两种：

  • win+R打开运行，输入regedit。进入注册表编辑器修改。
  • 运行后缀为.reg的文件

具体方法及道理：

Windows Registry Editor Version 5.00
[HKEY_CLASSES_ROOT\.md]
@="Typora.exe"
[HKEY_CLASSES_ROOT\.md\ShellNew]
"NullFile"=""
[HKEY_CLASSES_ROOT\Typora.exe]
@="Markdown"

@="Typora.exe" 代表的是指定.md文件的运行程序
@="Markdown" 代表的是右键时默认的文件名字，这样写新建为新建Markdown.md文件，而且右键菜单中显示 MarkDown

解释一下两种方式的对应关系:

[HKEY_CLASSES_ROOT.md] 是路径

这里@="Typora.exe" 相当于修改[HKEY_CLASSES_ROOT.md] 中项的值

@=ShellNew这个子项。所有有ShellNew子项的扩展名都会出现在【右键->新建】菜单中。

ShellNew各项值得意义：
  • "NullFile"="" 空文件
  • FileName 新建文件的模版文件
  • ItemName 可以修改编码方式（https://www.jianshu.com/p/e956eaae8828）
修改菜单排序:

修改后得右键菜单项会显示在：

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Discardable\PostSetup\ShellNew\Classes

所以，在这里可以修改右键新建文件菜单的排序

新建文件的命名是由打开方式的值决定的，比如

[HKEY_CLASSES_ROOT\.md]
@="Typora.md"```

如果是这么打开方式是这个，则在` Typora.md中修改值，就可以修改新建文件的默认名称。如下图，我的值是洛洛，新建文件名字便是 “新建洛洛.md”。

其他一些项：

Content Type和PerceivedType百度了一下，大概就是判断文件类型。

Windows Registry Editor Version 5.00 

[HKEY_CLASSES_ROOT\.md] 
@="Typora.md" 
"Content Type"="text/markdown" 
"PerceivedType"="text" 
[HKEY_CLASSES_ROOT\.md\ShellNew] 
"NullFile"=""

压死骆驼的最后一根稻草

又有一只骆驼倒下了，就在多加了一根稻草后。

我看着那奄奄一息的骆驼，他背上有如山的稻草，为何会被一根稻草压死。他似乎也看出了我的疑问。喘着最后的气息跟我说：

我也不知道为什么会倒下，只是感觉一整眩晕，就再也起不来了，我记得之前这里人烟还很少的时候，我就开始驮运货物了，我当时可是最勤快的那个，身体健硕。别的骆驼驮一百斤三天能到的路程，我驮一百五十斤，两天就到了。虽然，每次吃的伙食不怎么好，但喂的人还是会多给我扔一把干草。我们驮完货物，就在在那片水塘边歇息，那里有只叫声特别好听的鸟，我喜欢听她鸣叫。其他的骆驼说我如果是个人，肯定是个多情的诗人。

后来，人越来越多了，我去水塘边的时间越来越少，原先一周只需要驮一次货物，变成了两次，三次，四次，后来，便不按次数算了。以前，我是我们中走的最快的，慢慢似乎赶不上大家步伐了。我以为是大家都长高了，腿长了，走的快了，后来发现大家并没长，只是以前大家都在稳扎稳打的走，现在开始跑起来了，可是骆驼是不能跑的，要是一跑，抖起来，身上的稻草就容易撒。我不愿意跑，只是加快了步伐，可是我身上的鞭子痕迹越来越多。我后来发现，原来会检查干草的多少，现在只是检查你运的次数了。所以，那些跑的快的，有的还故意抖一抖身上草，都被奖励了各种吃的。

以前我会花两三天的工作去听那只鸟，后来一个月偶尔去一次，再后来，就没有时间与精力去了。每次运完货物，就赶紧回家休息一会，随时会被拉出去在拉货物，不过我们的住的棚子比起之前干净了不少，但很多骆驼就是为了住进这种新的棚子才拼命拉货的。最近，几个之前还住在我们老棚子里的骆驼也搬过来了，我问他们，那边怎样了，他们说，几乎没几匹骆驼了。草地都被污染了，长出来的草黄黄的，他们吃还行，孩子们吃就会拉肚子。

那么老骆驼呢？我问道。

哎！他们长叹了一口气，许多老骆驼都死了，为了让小骆驼住进在这新的骆驼棚里，很多早已多年不拉货的老骆驼又重新戴上了枷锁，吃着发黄的干草，所以，死的比较多。

我也没有多问，因为又要拉货了，身上的货物越来越多，干草看起来比以前多了，可是总是感觉饿，后来有个同行的骆驼骂骂咧咧的咬断一根干草，里面原来是空心的。我驮的货物越来越多，喂的干草堆比一起大了不少，可我越来越饿，似乎腿脚也没以前快了。

也有一些骆驼似乎特别开心，他们跑的很快，故意会把身上的稻草抖下来吃掉，在他们面前的干草堆比我大了好几倍，他们本来就没驮多重的货物，加上偷吃背上的稻草，所以，他们的稻草每次都会剩下来，稻草堆越积越大，吸引了许多拉不动货物的母骆驼，可我知道，那些母骆驼最终都没吃多少稻草就被赶了出去，一个骆驼，每天能吃多少稻草呢！

我的腿脚越来越不便利了，我还是老老实实的驮着草，一步步的往前的走，可是之前我会抬头看看太阳，有时候还会看看云朵，后来，我就不抬头了，我不知道是因为抬头太累还是因为太阳太刺眼了，我低着头，步伐越来越重，脚下的沙粒开始模糊起来，变成黄色的一片。我开始有些害怕了，在这片我祖祖辈辈生活的沙漠里，我开始讨厌沙漠，一眼望不到头，看不见哪里才是黄色的边际，那个水塘的图像在我心中越来越模糊，我都快要忘了那优美的鸟鸣声。

不过耳旁传来了呼噜噜 的声音，我刚开始还以为是雷声，后来，我才发现是一条长长的大蛇，大蛇好长好长，我从来没有见过这么大的蛇，蛇背上还驮满了稻草。我有些开心，又有些心慌，开心的是有这么厉害的大蛇，我是不是以后可以少驮点货物了，担心的是，万一我不驮货物，家里的老骆驼是不是就要被赶出骆驼棚。

不过我的担心是多余，我身上的货物一点也没有减少，反而越来越多。我也没有被赶出骆驼棚，只是在棚子的东面，又新盖了汗多骆驼棚，这边的许多骆驼都陆陆续续的搬了过去，这边的草儿也开始变黄，前两天家里的老骆驼还叫我找一套枷锁，他说他还能驮点货物，好让小骆驼早点搬进新的骆驼棚，这里都没有小骆驼和他一起玩耍了。

今天又要早早的来驮货物了，为了让小骆驼有些小伙伴。我让他们又加了一捆稻草，我吃了点昨天剩的额干草，里面的空心又粗了好多。我听到两个喂骆驼的闲聊，一个说有火车了，还为啥要这些骆驼，另一个说，不要干嘛，难道让他们去水塘边躺着？而且，没有骆驼，咋俩喂火车去？

后来，他们又聊到那边新盖的骆驼棚，然后，看了一眼我，说，像这一只就没机会住进去了，那里可是要拉够一万趟才能进去了，这只又蠢又懒，跑都跑不起来，下辈子看能拉够一万趟不!

然后，顺手又把地上掉落的一根稻草捡起来，扔到了我的背上，我感觉眼前一黑，就再也站不起来了。

他说完，抬头看了太阳，然后看着西北方，就这样死掉了。我知道，西北的角落曾经有一个水塘，那里有一只叫声特别好听的鸟，后来被抓住，不知卖到哪里去了，那个水塘也干涸了。现在用来填埋垃圾。

一个骆驼死掉了，会有专门的人来处理，这可不是件小事情，社会上传的沸沸扬扬，有人说落于哦是被毒死的，要彻查此事。有人说要给骆驼减负，骆驼们太幸苦了。还有人说，是喂骆驼的克扣骆驼的食材，骆驼被活活饿死。后来，传的更加离奇，说这只骆驼暗恋别人家的骆驼，看到地上掉落的稻草里里有母骆驼的毛发，睹物生情，悲伤而死。有更甚者，还专门出了一本书，记载道听途说的骆驼的故事，名字叫做骆驼样子。

骆驼死的第二天，来了一个西装革履的人，他走到骆驼的尸体前，象征性得到蹲下去看一眼，然后，捡起地上掉落的一根稻草，说这就是压死骆驼的最后一根稻草。勘察完了现场，他还给我们做了一场激动人心的演讲，说在西边又修个一个豪华的小镇，只要我们努力干，几百年后就可以去镇子里幸福生活了。

后来，那根稻草被收藏，被拍卖，一度达到了几十亿高价。再后来，被收藏在全球最大的博物馆。成为了博物馆的镇馆之宝。

-王校长落马记

王校长落马了。

这是校长秘书刚刚打电话告诉我的，我蹑手蹑脚的起来，没有敢打扰旁边熟睡的情人，稍稍收拾了一下，扔下五千块钱，便打车去了单位。

当我坐在办公桌前的时候，天已经微微亮了，东方既白，像翻着肚皮的死鱼。我从抽屉里拿出一盒烟，抽出一根，却怎么也找不到打火机。

“这次还是狼来了？”我推开椅子，在不大 的办公室里来回踱着。这办公室却是有些寒酸，上次老家的亲戚来找我，他一进门后的，眼神瞬间由期待变得失望。这也不怪他，谁能想到堂堂一个市的文化局局长，办公室竟然这么寒酸。也不是因为我们没钱，正是因为太有钱了，所以，才要显得没钱。这是王校长的政治智慧。王校长特别喜欢这一套，每次政治作秀都能够博得群众的好评，电视台的采访，他就待在自己那间简朴的小办公室里，用带着口音的普通话，说着我前天晚上写好的稿子。稍微有些分辨能力的人都会感到虚假，但是老百姓就吃这一套。市政府前段时间建立一座新的政府楼，许多下面的部门都搬过去了，就是市长和市委主要成员还留在这九十年代的老楼里。外面马上就传出响应，市长真是不忘初心，艰苦奋斗，把优良的条件都留给后生。我是知道的，他一个月在他那间破办公室里待不了两个小时，而那两个小时都是为了应付市电视台每月的市长连线。我有时候在想，我们市长如果进军演艺圈，一定是个实力派演员。可比那些电视上的小鲜肉强多了。其实不能说市长没有进军演艺圈，只是他的进军目标不是当上演员，而是上演员。

想到这里，我就不再犹豫了。马上拿起手机，拨通了市台台长的电话。

节目反响很好，你别看这个四十多岁的半老徐娘，正经大学都没上过，可是搞宣传工作真的有一套。还有那个演农民的演员，别说，长的像农民就算了，连说话的方式都演的像。给人一种真实感。工作组效率也是杠杠的，在下午六点半的市政新闻里面，就已经播出了。而当时我正站在王校长的病床前。

财主家要选一个算账的伙计，于是财主家的门庭开始繁忙起来。来来往往的人都要踩着后脚跟了，财主家的门槛都被磨低了一大截。

这活是个肥差，别看也是要幸幸苦苦加班加点的干。但是能跟掌柜的熟，还掌握着账薄。油水可不少，所以，大家都把头消尖了，打算挤进去。

先是考试，一人发一张白纸，让大家写数，从一写到一万，不能有错。选前几个写完的。结果选出来了，大家并不是很满意，刘副掌柜的侄子和钱大掌柜的外甥都入选了。平时这两个人都不学无术，怎么还会有如此才能。

不过也没有人发表啥意见，只是钱大掌柜和刘副掌柜家里的灯这几天晚上都亮到很晚。

再接着就是开会，面试。几大掌柜端坐在堂上，高高在上。

一个人的看法总归是有失偏颇的，所以，需要大家一起帮忙去看看清楚。不同人有不同的看法，有些人摸到了老虎的屁股，有些人摸到了老虎的头，有些人被老虎咬了一口，大家的看法合起来，或许能够成为一个完整的老虎，而不是很多人一眼看上去可爱的大猫。不同人有不同的想法观点，有分歧，大家讨论，说出来，总结一下，便更全面了。可是，大家都不是说话的人，而是听话的人。选个人，喜欢选个听自己话的人，而又没几个人去反对。于是整个集体中又多了一个听话的人，下一次再选人，赞成的声音又多了一份，反对的声音就又弱了一点。偶尔有几个愿意说自己意见的人，可每次意见好像大家都不赞同，最后，就开始怀疑自己，缄口不言了。人总归是人，哪有那么大的勇气，一直和大流唱反调，那些棱角分明的石头，在水中，不都磨得圆圆滑滑的了。

财主家的狗不叫了，因为来的是提满腊肉的贵人。若是门口来了要饭的叫花子，他还是吠得厉害，要是有人来门口大骂，他就恨不得把自己得牙齿拔下来，扎住骂人的人的嘴。财主家的狗是条好狗，只是他最近不怎么咬人了，因为听说，财主的院子外，又筑了一道高墙，这下，那些只说话不拎着东西的人，连财主家的大门都不到了。

后来的事情就不得而知了，人们只是注意到，高墙里进进出出的人越来越少，狗也渐渐不叫了。

我见过有一个人在领导面前乖乖的像个等着被训的孩子。

财主家的故事

财主家要选一个算账的伙计，于是财主家的门庭开始繁忙起来。来来往往的人都要踩着后脚跟了，财主家的门槛都被磨低了一大截。

这活是个肥差，别看也是要幸幸苦苦加班加点的干。但是能跟掌柜的熟，还掌握着账薄。油水可不少，所以，大家都把头消尖了，打算挤进去。

先是考试，一人发一张白纸，让大家写数，从一写到一万，不能有错。说是选前几个写完的。结果选出来了，大家发现，刘副掌柜的打字不识侄子和钱大掌柜数不清一二的外甥都入选了。顺便还获得了年度最佳书法奖。

不过也没有人发表啥意见，只是钱大掌柜和刘副掌柜家里的灯这几天晚上都亮到很晚。

再接着就是开会，面试。几大掌柜端坐在堂上，高高在上。彷佛庙堂里的几尊佛像，接受人们的朝拜。会议很成功，大家都很满意。最终刘副掌柜的侄子成功加入了啃得起豪华套餐。当然，这位大字不识一个的侄子，入选的理由合情合理。具有创新思维，敢做敢当。

入职的当天，财主家当铺门前热闹非凡，到处都是赞扬的声音。不过这大侄子也倒是敢做敢当，账目记得好坏不知道，别人欠的钱倒是要回来好几笔，只是镇上的人口失踪了几个。

后来，财主家的狗叫了好几天，据说一个穿着长跑的人，跑到财主家门口，满口什么之乎者也。拉着刘掌柜的衣角，问刘掌柜老虎长什么样子？

再后来，财主家的狗也不叫了，来的都是提腊肉的人。但若是门口来了要饭的叫花子，他还是吠得厉害，要是有人来门口之乎者也，他就恨不得把自己得牙齿拔下来，扎住那人的嘴。

之后的事情就不得而知了，财主的院子外，又筑了一道高墙，这下，那些只会之乎者也不拎着东西的人，连财主家的大门都不到了。人们只是注意到，高墙里进进出出的人越来越少，狗也渐渐不叫了。
